# ChairBedSystem using qb-target and qb-radialmenu

**Installation:**

```lua
[1] =  {
        id = 'bed:up',
        title = 'Stand Up',
        icon = 'hand-point-up',
        type = 'client',
        event = 'bed:up',
        shouldClose = true
       },
```
